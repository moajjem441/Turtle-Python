# 🐢 Python Turtle Graphics Cheat Sheet

টার্টল গ্রাফিক্সে কাজ করার জন্য প্রয়োজনীয় কিছু বেসিক কমান্ড এবং সুষম বহুভুজ (Regular Polygon) আঁকার গাণিতিক নিয়ম নিচে দেওয়া হলো।

### 🛠️ বেসিক কমান্ডসমূহ (Basic Commands)

| Command | Description |
| :--- | :--- |
| `t.forward(100)` | সামনে ১০০ পিক্সেল যাবে |
| `t.backward(100)` | পিছনে যাবে |
| `t.left(90)` | বামে ৯০° কোণে ঘুরবে |
| `t.right(90)` | ডানে ৯০° কোণে ঘুরবে |
| `t.penup()` | কলম উপরে তুলবে (আঁকা বন্ধ হবে) |
| `t.pendown()` | কলম নিচে নামাবে (আবার আঁকা শুরু হবে) |
| `t.circle(50)` | ৫০ রেডিয়াসের বৃত্ত আঁকবে |
| `t.speed(1)` | গতি সবথেকে কম (Slow) |
| `t.speed(10)` | গতি বেশি (Fast) |

---

### 📐 সুষম বহুভুজ আঁকার সূত্র

যেকোনো সুষম বহুভুজ আঁকার জন্য টার্টলকে মোট ৩৬০° ঘুরতে হয়। বাহুর সংখ্যা অনুযায়ী ঘোরার কোণ বের করার সূত্র:

$$Angle = \frac{360}{\text{বাহুর সংখ্যা}}$$

#### বিভিন্ন আকৃতির হিসাব:

| আকৃতির নাম | বাহুর সংখ্যা (n) | কোণের হিসাব (360/n) | কোড (Angle) |
| :--- | :---: | :---: | :--- |
| **ত্রিভুজ** | ৩ | $120^\circ$ | `t.left(120)` |
| **বর্গক্ষেত্র** | ৪ | $90^\circ$ | `t.left(90)` |
| **পঞ্চভুজ** | ৫ | $72^\circ$ | `t.left(72)` |
| **ষড়ভুজ** | ৬ | $60^\circ$ | `t.left(60)` |
| **বৃত্ত** | ৩৬০ | $1^\circ$ | `t.left(1)` |
